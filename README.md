# Student Management System

A Next.js 14 student management dashboard with login/register, student and attendance management, online exam UI, and results reporting.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Axios

## Available scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`

## App pages

- `/login`
- `/dashboard`
- `/students`
- `/attendance`
- `/exam`
- `/results`

## Notes

This starter app uses a client-side auth flow with a simple API service layer in `lib/api.ts`. It is designed as a clean, responsive codebase for a student management system.
